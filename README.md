Данный репозиторий содержит программную реализацию моделирования распространения лазерных пучков Гаусса-Лагерра (ГЛ) через турбулентную атмосферу с использованием дифракционных оптических элементов (ДОЭ) для управления поперечно-модовым составом пучка.

## Описание

Проект решает следующие задачи:
- Генерация пучков Лагерра-Гаусса с различными топологическими зарядами и радиальными индексами
- Синтез ДОЭ для преобразования гауссова пучка в заданную моду ГЛ с помощью метода введения дополнительной области
- Моделирование распространения пучков через турбулентную атмосферу с использованием метода фазовых экранов
- Оценка качества преобразования и влияния турбулентности на модовый состав

## Теоретическая основа

### Пучки Лагерра-Гаусса
Комплексная амплитуда пучка Лагерра-Гаусса описывается выражением:

$$E_{LG}(r,\phi) = \frac{1}{\sigma_0} \sqrt{\frac{2n!}{\pi(n+|m|)!}} \left(\frac{\sqrt{2}r}{\sigma_0}\right)^{|m|} L_n^{|m|}\left(\frac{2r^2}{\sigma_0^2}\right) \exp(im\phi) \exp(\frac{-r^2}{\sigma_0^2})$$

где:
- $n$ - радиальный индекс
- $m$ - азимутальный индекс (топологический заряд)
- $\sigma_0$ - радиус перетяжки
- $L_n^{|m|}$ - обобщенный полином Лагерра

### Дифракционный оптический элемент
ДОЭ рассчитывается методом дополнительной области [Управление поперечно-модовым составом когерентного излучения / Павельев В.С. [и др.]. Самара: Издательство Самарского государственного аэрокосмического университета. 2007. 192 с.], который обеспечивает формирование заданного распределения комплексной амплитуды в фокальной плоскости.

### Турбулентность
Моделирование турбулентности основано на методе фон Кармана [J. D. Schmidt, Numerical simulation of optical wave propagation with examples in MATLAB. Bellingham: SPIE Press, 2010] с использованием модели фазовых экранов, предоставляемой функцией aotools.turbulence.phasescreen.
ft_phase_screen (r0, N, delta, L0, l0) библиотеки aotools. Параметры:
- Параметр Фрида r0 - характеризует силу турбулентности
- Внешний масштаб L0 - определяет размер наибольших неоднородностей
- Внутренний масштаб l0 - определяет размер наименьших неоднородностей
